# AAB Project

This repository contains code and notebooks for the Automated Wastewater Pipeline project.

## Structure
- `src/` — Python scripts for running the pipeline
- `notebooks/` — Jupyter notebooks for data exploration and preprocessing
- `data/` — (empty, or small sample data only; real/large data is ignored)
- `results/` — (empty; results are ignored)

## Usage
1. Place your FASTQ files in a data directory (not tracked by git).
2. Run the pipeline scripts in `src/`.
3. See notebooks in `notebooks/` for analysis and exploration.

## Requirements
See `requirements.txt` for Python dependencies.

## Notes
- Large data and results are not tracked in git.
- Sensitive or private data should not be committed.
